Introdução

O Flexbox, ou Flexible Box Layout, é uma especificação do CSS projetada para criar layouts mais eficientes e responsivos. Seu objetivo é simplificar o processo de alinhar e distribuir o espaço entre itens em um container, independentemente das dimensões desses itens, que podem ser desconhecidas ou variar dinamicamente.
