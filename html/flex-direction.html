Flex-Direction

Define a direção dos flex itens. Por padrão ele é row (linha), por isso quando o display: flex; é adicionado, os elementos ficam em linha, um do lado do outro.
A mudança de row para column geralmente acontece quando estamos definindo os estilos em media queries para o mobile.
Assim você garante que o conteúdo seja apresentado em coluna única.

Alguns exemplos:

flex-direction: row;

1
2
3

flex-direction: row-reverse;

1
2
3

flex-direction: column;

1
2
3

flex-direction: column-reverse;

1
2
3
